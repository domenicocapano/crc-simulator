Software Description

The software “CRC Simulator — Cyclic Redundancy Check” is an educational simulator developed to support the teaching and learning of error detection using the CRC (Cyclic Redundancy Check) algorithm.

The simulator was primarily designed for Computer Networks students in Italian Technical Institutes (ITIS), but it can also be used in introductory courses on computer networks and digital communication systems.

Main Features

- CRC computation on the transmitter side
- Binary modulo-2 division using XOR
- Step-by-step visualization of the division process
- Construction of the transmitted frame
- Receiver-side verification
- Single-bit error simulation
- Burst error simulation
- Interactive dynamic board
- Step-by-step mode
- Smartphone/tablet support
- Fast mode (“Skip to End”)
- Mathematical theory modal window
- WCAG-AA accessibility support
- Result export

Educational Objectives

The simulator allows students to:

- understand the mathematical operation of CRC;
- understand the role of XOR in modulo-2 division;
- visualize the construction of the transmitted frame;
- understand error detection in the Data Link layer;
- distinguish between:
  - original message;
  - extended message;
  - CRC;
  - transmitted frame;
- understand the concept of integrity checking in network protocols.

Descrizione del software

Il software “Simulatore CRC — Codici di Ridondanza Ciclica” è un simulatore didattico sviluppato per supportare l’insegnamento e l’apprendimento del controllo degli errori mediante algoritmo CRC (Cyclic Redundancy Check).

Il simulatore è stato progettato principalmente per studenti di Sistemi e Reti degli Istituti Tecnici Industriali (ITIS), ma può essere utilizzato anche in corsi introduttivi sulle reti di calcolatori e sui sistemi di comunicazione digitale.

Funzionalità principali
Calcolo del CRC lato trasmettitore
Divisione binaria modulo 2 mediante XOR
Visualizzazione passo-passo della divisione
Costruzione del frame trasmesso
Verifica lato ricevente
Simulazione di errori singoli
Simulazione di burst error
Lavagna dinamica interattiva
Modalità passo-passo
Supporto smartphone/tablet
Modalità rapida (“Vai alla fine”)
Modale matematica teorica
Accessibilità WCAG-AA
Esportazione dei risultati
Obiettivi didattici

Il simulatore consente agli studenti di:

comprendere il funzionamento matematico del CRC;
comprendere il ruolo dello XOR nella divisione modulo 2;
visualizzare la costruzione del frame trasmesso;
comprendere la rilevazione degli errori nel livello Data Link;
distinguere tra:
messaggio originale;
messaggio esteso;
CRC;
frame trasmesso;
comprendere il concetto di controllo di integrità nei protocolli di rete.
